Random Number Generator (Tkinter)

A simple and user-friendly random number generator built with Python and Tkinter.
Supports integers and decimals, custom precision, multilingual interface (English/Spanish), input normalization, result copying, and a session history with reloadable configurations.

Features

Integer and decimal number generation

Custom decimal precision

Automatic input normalization (supports comma or dot)

History of generated values

Reload previous configurations from history

Copy result to clipboard

English/Spanish interface

Error handling for invalid input

No external dependencies (standard library only)
